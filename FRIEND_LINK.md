# 友链交换说明

### 格式
```yaml
-
  name: # 称呼, 不会显示在页面上
  title: # 博客名称*
  site: # 博客链接*
  desc: # 博客描述*
  email: # 你的邮箱(可选)
  # 以下配置暂时被忽略, 以后可能会启用
  avatar: # 头像链接, 用于展示
  # 推荐通过注释来介绍自己
```

### 流程
1. fork 我的[博客仓库](https://github.com/ruan4261/ruan4261.github.io/)
2. 将你的信息根据上述格式 append 到`/src/_data/_friendlink.yml`文件中
3. 向我的博客仓库的 master 分支提交 pull request
4. 请注意及时在您的友链页面加上我哦^. ^
